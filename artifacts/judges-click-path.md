# ZK Dungeon: Judges Click Path

Please follow these steps to verify the live demo of ZK Dungeon on the Stellar Testnet.

### Prerequisites
- Install the [Freighter Browser Extension](https://www.freighter.app/).
- Set Freighter to the **Testnet** network.
- Ensure your testnet wallet is funded (you can use the in-app Friendbot button if needed).

### Step-by-Step Reproduction
1. **Navigate to the App**
   Open your browser and navigate to the live environment:
   `https://smol-fe-7jl.pages.dev/labs/the-farm`

2. **Enter ZK Dungeon**
   Scroll or click through to the **ZK Dungeon** experiment tile. The URL should update to `/labs/the-farm/zkdungeon`.

3. **Enable Hackathon Mode**
   At the bottom or top of the title screen, look for the **"Hackathon Mode"** toggle and ensure it is switched **ON**. This directs the application to use the Stellar Testnet rather than mainnet, bypassing standard passkey limits for easier judging.

4. **Connect Wallet & Play Solo**
   - Click the **"Play Solo"** button.
   - A modal will appear prompting you to connect a wallet via Stellar Wallets Kit.
   - Select **Freighter** and approve the connection request in the extension popup.

5. **Interact with the Blockchain**
   - Upon entering the dungeon and completing a progression step, the app will execute its core logic.
   - If prompted by Freighter, sign the transaction. This validates the smart contract execution (`start_game` / proof submission / `end_game`).
   - *Note: Our multiplayer matching service is currently being upgraded, so the solo mode demonstrates the primary wallet+contract integration loop.*

6. **Verify the Transaction**
   - You can copy your wallet address and paste it into [Stellar Expert (Testnet)](https://testnet.stellar.expert/) to see the smart contract invocations generated by your gameplay.
